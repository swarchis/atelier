# La Guia / Grainline Web

A Vite + React frontend shell for a production OS for independent clothing brands.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes for collaborators

- Do not commit `node_modules`; install dependencies with `npm install`.
- Keep `package-lock.json` committed so everyone installs the same dependency versions.
- Use branches or pull requests when making larger changes.
